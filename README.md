# Maze Escape Game

A fun maze game with 10 levels, built with Kivy for Android!

## Features
- 🎮 Touch controls (swipe or buttons)
- 🐍 Smart enemy snake with AI pathfinding
- 💡 Hint system
- 📱 Mobile-optimized interface
- 🏆 10 progressive levels

## Download
Download the latest APK from [Releases](https://github.com/Kceecalvin/maze-escape-game/actions) (Artifacts section)

## How to Play
1. Swipe or tap arrow buttons to move
2. Reach the red exit square
3. Avoid the green snake!
4. Use hints if you get stuck

Built with ❤️ using Kivy
