========================================
   GITHUB ACTIONS - EASIEST METHOD
========================================

This is the FASTEST way to get your APK!

Why GitHub Actions?
  ✓ No local setup needed
  ✓ Works on any computer
  ✓ Free for public repositories
  ✓ Build happens in the cloud
  ✓ Takes 20-30 minutes, runs automatically

========================================
STEP-BY-STEP INSTRUCTIONS:
========================================

1. INITIALIZE GIT (if not already done)
   Run these commands in PowerShell:

   git init
   git add .
   git commit -m "Initial commit - Maze Escape Game"

2. CREATE GITHUB REPOSITORY
   a. Go to: https://github.com/new
   b. Repository name: maze-escape-game
   c. Choose Public (for free builds)
   d. Click Create repository

3. PUSH YOUR CODE TO GITHUB
   GitHub will show you commands like:

   git remote add origin https://github.com/YOUR_USERNAME/maze-escape-game.git
   git branch -M main
   git push -u origin main

   Note: Replace YOUR_USERNAME with your actual GitHub username!

4. TRIGGER THE BUILD
   a. Go to your repository on GitHub
   b. Click Actions tab at the top
   c. Click Build Android APK on the left
   d. Click Run workflow button (top right)
   e. Click green Run workflow button

5. WAIT FOR BUILD (20-30 minutes)
   You can close your computer and come back later!

6. DOWNLOAD YOUR APK
   a. Click on the completed workflow run
   b. Scroll to Artifacts section at the bottom
   c. Click maze-escape-apk to download
   d. Extract the ZIP file
   e. Install the APK on your Android device!
